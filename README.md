Bem-vindo ao repositório oficial do EcoChuva! Este projeto é uma plataforma web interativa
focada em sustentabilidade, educação ambiental e cálculo prático para o reuso de recursos hídricos residenciais
O sistema foi desenvolvido para ser simples, direto e acessível a qualquer pessoa que deseja economizar água e reduzir o impacto ambiental de forma prática.
O EcoChuva une tecnologia e conscientização ambiental em uma aplicação web única.
Ele permite que o usuário crie uma conta de forma segura (com dados salvos diretamente visualize ideias inteligentes para reaproveitar a água pluvial e calcule instantaneamente
o potencial de captação do seu telhado, acompanhando o resultado através de uma simulação visual animada em tempo real.

Página de Capa e Autenticação: Uma interface de apresentação elegante com sistema de login
e cadastro persistente utilizando o armazenamento interno do navegador (localStorage). Suas credenciais ficam salvas de verdade!

Plano de Ação Ecológico (Pós-Login): Um guia detalhado que orienta o usuário sobre as melhores formas de utilizar a água coletada
(como limpeza geral, descargas sanitárias, irrigação de jardins e lavagem de veículos).

Simulador de Captação Simplificado: Cálculo matemático preciso baseado na área útil
horizontal do telhado (Largura $\times$ Comprimento) e nos índices pluviométricos (mm/h), eliminando a necessidade de dados complexos de profundidade estrutural.

Animação em Tempo Real: Um tanque interativo que enche de forma dinâmica de acordo com o percentual de armazenamento atingido após a simulação.

Sistema de Avaliação: Interação social que permite ao usuário classificar a utilidade da ferramenta em uma escala de 1 a 5 estrelas.
